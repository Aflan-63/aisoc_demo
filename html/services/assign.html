How to Assign a Service


When a service is assigned to an user, the user will have the access to the privileges of the service. User can be assigned or reassigned to a service.

Assign
  • 1. At first select a service from the dropdown, you will see all the services from the list.
  • 2. Select a service, then select an user from the list.
  • 3. Click on the assign button to complete the assignment.

    • Re-assign
  • 1. At first select a service from the dropdown, you will see all the services from the list.
  • 2. Select a service, then the already assigned user will be shown on the dropdown, select a different user from the drop down. It will auto remove the previous assigned user.
  • 3. Click on the assign button to complete the re-assignment.
